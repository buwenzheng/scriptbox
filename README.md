# ScriptBox 🧰

自用脚本box

## 📁 项目结构

```
scriptbox/
├── javascript/           # JavaScript 用户脚本
│   ├── linuxdoAutoReadAndLike.js    # Linux.do 自动阅读和点赞脚本
│   └── LDStatus.js                  # Linux.do 信任级别状态显示脚本
├── sh/                   # Shell 脚本
│   └── update_clash_conf.sh         # Clash 配置文件自动更新脚本
└── README.md
```

## 🚀 功能特性

### JavaScript 用户脚本

#### 1. Linux.do 自动阅读和点赞脚本 (`linuxdoAutoReadAndLike.js`)

专为 [Linux.do](https://linux.do) 社区设计的自动化脚本，基于 Discourse 论坛系统。

**核心功能：**
- 🔄 **自动滚动阅读**: 模拟用户阅读行为，定时自动滚动页面
- ❤️ **智能随机点赞**: 随机选择帖子进行点赞，避免被检测为机器行为
- 👁️ **页面可见性处理**: 解决 Windows 系统下页面失焦时滚动暂停的问题
- ⚙️ **灵活配置**: 可调节滚动间隔、点赞频率等参数
- 🎛️ **直观控制面板**: 提供友好的控制界面

**技术亮点：**
- 使用 Web Workers 确保后台运行
- 重写 Page Visibility API 绕过 Discourse 的懒加载限制
- 智能去重避免重复点赞
- 兼容多种点赞按钮样式（原生 + 插件）

#### 2. Linux.do 状态显示脚本 (`LDStatus.js`)

显示 Linux.do 用户信任级别进度的实用工具。

**功能特性：**
- 📊 实时显示信任级别进度
- 🎨 支持亮色/暗色主题切换
- 📱 可拖拽的悬浮面板
- 🔄 自动更新检测

### Shell 脚本

#### Clash 配置自动更新脚本 (`update_clash_conf.sh`)

用于自动更新 Clash 代理配置的 Bash 脚本。

**功能特性：**
- 🔄 自动从订阅链接下载最新配置
- ✅ 智能文件校验（格式、内容完整性）
- 🔍 配置变更检测，避免无意义更新

### Clash 脚本常见问题

**Q: 提示"curl 命令执行失败"？**
A: 检查网络连接和订阅链接是否正确。

**Q: 提示"文件为空"？**
A: 订阅链接可能已过期或流量耗尽，请联系服务提供商。

**Q: 提示"权限不足"？**
A: 确保脚本有写入目标目录的权限，或使用 `sudo` 运行。
